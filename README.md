# FolderSnapshot

Создает "снимки" указанного каталога, снимки хранятся в отдельных файлах и отображаются в программе. Благодаря "снимкам" списка файлов указанных папок можно очистить указанную папку на момент "снимка". Имеется множественная защита от ошибок пользователя.
